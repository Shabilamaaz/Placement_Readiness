# 🚀 Placement Prediction System

A Flask-based web application that helps students analyze their placement readiness using machine learning and smart suggestions.

---

## 🔥 Features

- 🎯 Placement Prediction (ML based)
- 📊 Performance Graph
- 🧠 Smart Suggestions
- 📅 Weekly Study Plan
- 📄 Resume Analyzer (Score + Tips)
- 🏆 Leaderboard (Top Users)
- 👤 Profile Page (Stats & Progress)
- 🏢 Company-wise Preparation (TCS, Infosys, Amazon)
- 🗂️ Prediction History (Delete & Clear)
- 📥 Download Report (PDF)
- 📧 Email Suggestions

---

## 🛠️ Tech Stack

- Python (Flask)
- SQLite Database
- HTML, CSS (Dark UI)
- Machine Learning (Logistic Regression / Scikit-learn)

---

## 📊 How it Works

1. User enters:
   - CGPA
   - Aptitude
   - Coding
   - Communication
   - Projects & Internships  

2. Model predicts:
   - Placement Result (Placed / Not Placed)
   - Chance Percentage  

3. System provides:
   - Smart Suggestions  
   - Weekly Study Plan  
   - Performance Graph

## ▶️ How to Run Project

### 1. Clone Repository
```bash
git clone https://github.com/Shabilamaaz/Placement_Readiness.git
cd Placement_Readiness
```
### 2. Install Dependencies
```bash
pip install flask scikit-learn pandas matplotlib reportlab
```
### 3. Run Application
```bash
python app/app.py
```
4. Open in Browser
```bash
http://127.0.0.1:5000
```
